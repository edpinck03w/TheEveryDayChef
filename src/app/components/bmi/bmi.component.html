BMI Calculator - Strings from a stream

Take legend strings and colors from a stream

  • Taking legend information from a stream might seem a bit of overhead, but in a real application they actually come from a service call or a store, so lets keep them so.
  • withLatestFrom is just fine to combine the strings if the are constant throughout app lifespan. For a localizable app though, user might want to switch the language at any time and see themchange immediately. in order to be able to react to this change, we should use combineLatest then.
